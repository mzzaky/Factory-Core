Updates & Improvements
  • Marketplace Item Registration: Simplified the item registration system in the Marketplace Sell Items menu, replacing the item list with a direct click mechanism from the player's inventory.
  • Suggested Price Attribute: Added suggested_price attribute to all resources in resources.yml as a dynamic reference for selling prices in the Marketplace.
  • Marketplace Listing Limit: Changed the maximum number of items a player can sell from 10 to 3 in the marketplace feature.
  • Business Expansion Research: Added new "business_expansion" research to increase the maximum marketplace listing capacity by +1 per level.
  • Real-time Expiration: Added real-time expiration time remaining information for items sold in the Marketplace My Listings menu.
  • Market Stability Research: Added new "market_stability_regulation" research to increase the expiration time of player sales in the marketplace by +8 hours per level.
  • New Employee NPCs: Added more NPC variants to the Employee shop along with their unique templates (Logistics Manager, Quality Inspector, Assembly Expert).
-
Bug Fixes
  • Invoice Center Tax Display: Fixed the Invoice Center GUI to correctly display factory tax details.
  • Invoice Center Processing: Fixed the filter, sort, Single Pay, and Pay All features in the Invoice Center to properly process the tax system.
  • Marketplace Duplication Exploit: Fixed an item duplication exploit in the Marketplace when canceling a sell queue. Items are now completely removed from the inventory upon registration.

Gameplay Changes
  • NPC Interaction Rework: Left-click now opens the Output Storage, and right-click opens the main Factory GUI.
  • Default Height Offset: Adjusted from 2.3 to 1.3 for improved NPC placement.
  • Storage on Upgrade: Removed the automatic increase of input/output storage slots when a factory is upgraded — storage scaling is no longer tied to upgrade level.
+
New Mechanics
  • Upgrade Sound Effect: A sound effect now plays when a factory is successfully upgraded.
  • Upgrade Confirmation GUI: A confirmation dialog is now shown before a factory upgrade is executed, preventing accidental upgrades.
  • Timed Upgrades (time_upgrade): Factory upgrades now require a configurable duration (in seconds) to complete — instant upgrades are no longer possible. Adjustable via config.yml.
  • Factory Ownership Limit: Each player can now own a maximum of 3 factories by default. This limit is configurable in config.yml.
  • Recipe Money Cost: A money cost requirement has been added to every available production recipe.
+
Factory Research System
  • 6 Research Technologies available for factories to unlock and level up:
    • Fiscal Optimization — Reduces tax costs.
    • AI Workforce Integration — Reduces employee salary costs.
    • Nano-Construction Framework — Reduces factory upgrade time.
    • Advanced Machine Technology — Reduces item production time.
    • Operational Cost Efficiency — Reduces item production cost.
    • Industrial Mastery — Adds +1 to the factory ownership limit per level.
  • Research Costs & Duration: Each research requires a significant money investment and a completion time (default: 1 day).
  • Research Center GUI: A dedicated custom GUI linked to the Hub GUI for managing all research activities.
  • research.yml Config: New configuration file for full control over all research parameters.
  • Dynamic GUI Buffs Display: Research buffs are reflected in real-time across multiple GUIs:
    • Recipe Selection GUI shows updated production time/cost from advanced_machine_technology and operational_cost_efficiency.
    • Upgrade GUI shows real-time upgrade time reduction from nano_construction_framework.
    • My Factories GUI shows adjusted factory ownership limit from industrial_mastery.
    • Tax Center GUI shows real-time tax deduction calculations from fiscal_optimization.
    • Invoice GUI shows real-time salary cost reductions from ai_workforce_integration.
+
New Admin Commands
  • /fc admin research upgrade <factory_id> <research_id> — Manually advance a factory's research by one level.
  • /fc admin research set <factory_id> <research_id> <level> — Manually set a factory's research to a specific level.

Factory System
  • 4 Factory Types: Steel Mill ($100,000), Refinery ($150,000), Workshop ($200,000), and Advanced Factory ($250,000) — each with unique input/output chains.
  • Factory Ownership System: Players can buy and sell factories. Sell price is 50% of the purchase price (configurable).
  • WorldGuard Region Binding: Each factory is tied to a defined WorldGuard region for area management.
  • Factory Level System (1–5): Upgradeable factories with per-level bonuses — 5% production cost reduction, 10% time reduction, +9 storage slots, and +2.5% tax per level.
  • Factory Status Tracking: Real-time status display: RUNNING, STOPPED, OUT OF MACHINE PARTS.
+
Production System
  • 16 Predefined Recipes across all factory types: 2 for Steel Mill, 3 for Refinery, 5 for Workshop, and 6 for Advanced Factory.
  • BossBar Progress Indicator: Real-time production progress shown via configurable BossBar (color and style configurable).
  • Production Queue System: Up to 5 queued production tasks per factory.
  • Console Command Execution: Custom console commands triggered upon production completion (e.g., economy rewards).
+
Resource System
  • 21 Resources in 5 Categories:
    • Raw Resources (6): Iron Ore, Coal Ore, Copper Ore, Gold Ore, Diamond Ore, Redstone Dust
    • Advanced Resources (4): Steel Ingot, Refined Copper, Gold Alloy, Carbon Fiber
    • Final Resources (4): Advanced Weapon ($5,000), Advanced Armor ($4,500), Electronic Component ($3,000), Luxury Item ($10,000)
    • Fuel (3): Basic Fuel, Refined Fuel, Premium Fuel
    • Machine Parts (5): Basic Gear, Advanced Gear, Engine Part, Circuit Board, Hydraulic System
  • Custom Model Data Support for all 21 resources (IDs 1001–5005).
  • Enchantment Glow Effect for rare items (Gold Alloy, Diamond Ore, Premium Fuel, Circuit Board, Hydraulic System, and all Final Resources).
+
Economy & Tax System
  • Full Vault Integration for all economic transactions (buy, sell, tax, salary, marketplace).
  • Tax System: 5% of factory price per 3 Minecraft days (144,000 ticks). +2.5% per factory level. 7-day grace period before overdue. 5% late fee on overdue taxes.
  • Salary System: 1% of factory price per 1 Minecraft day (24,000 ticks), automatically billed as invoices.
  • Invoice Management: Auto-generated invoices for tax and salary. Supports individual or bulk payment. Factory is disabled on unpaid overdue invoices.
+
NPC Employee System
  • 5 Employee Tiers available in the Employee Shop:
    • Apprentice Worker — $500 (salary $50): −5% production time
    • Skilled Craftsman — $1,500 (salary $120): −15% production time
    • Steel Forger — $2,500 (salary $200): −20% production time
    • Refinery Operator — $3,500 (salary $280): −25% production time
    • Master Engineer — $6,000 (salary $450): −35% production time
  • Advanced NPC Behavior: Head tracking (6-block radius), idle animations with smooth interpolation, unique particle effects per factory type, hologram display, ambient/interaction sounds, body rotation, invulnerable & AI-disabled.
  • 4 NPC Templates matching factory types: Steel Mill Worker (ARMORER/PLAINS), Refinery Operator (TOOLSMITH/DESERT), Workshop Engineer (WEAPONSMITH/SNOW), Advanced Factory Manager (LIBRARIAN/JUNGLE).
  • Auto-Respawn System: Periodic check and re-spawn of missing NPCs to ensure persistence.
+
GUI System
  • 17 Interactive GUI Screens: HubGUI, MainMenuGUI, FactoryBrowseGUI, MyFactoriesGUI, RecipeSelectionGUI, RecipeConfirmationGUI, StorageGUI, OutputStorageGUI, InvoiceGUI, InvoiceCenterGUI, TaxCenterGUI, UpgradeGUI, EmployeesCenterGUI, EmployeeShopGUI, MarketplaceGUI, HelpInfoGUI, FactoryGUI.
  • Configurable GUI Elements: Border item, back button, close button, confirm/cancel items — all configurable in config.yml.
  • Hub GUI serves as the central navigation point with player head display and quick stats.
+
Marketplace System
  • Player-to-Player Resource Trading via an in-game marketplace GUI.
  • Marketplace Tax: 5% deducted from the seller on each sale (configurable).
  • Listing Rules: Max 10 listings per player. Listings expire after 7 days (168 hours). Price range: $0.01 – $1,000,000.
  • Earnings Collection System: Sellers collect earnings from a dedicated earnings panel.
+
Storage System
  • Per-Factory Input & Output Storage: Separate storage for depositing inputs and collecting outputs.
  • Expandable Slots: Base 9 slots, +9 per factory level (max 54 slots at level 5).
  • Persistent YAML Storage: All inventory data is saved and restored across server restarts.
  • Item Validation: Only valid FactoryCore resources can be deposited into factory storage.
+
Command System
  • Main Command: /factorycore with alias /fc.
  • User Commands: /fc hub, browse, my, invoices, taxes, employees, market, buy, sell, info, gui, tp, help, version.
  • Admin Commands: /fc admin create/remove/list/info/reload/setowner/teleport/give, npc spawn/remove, tax, market.
  • Full Tab Completion support for all commands and subcommands.
+
Permission System
  • 12 Admin Permissions (default: OP): factorycore.admin.* — covering create, remove, list, info, reload, setowner, teleport, give, npc, tax, and market.
  • 15 User Permissions (default: true): factorycore.use grants access to buy, sell, info, gui, hub, browse, myfactories, invoices, taxes, employees, marketplace, teleport, help, and version.
  • Wildcard Node: factorycore.* grants all permissions at once.
+
PlaceholderAPI Support
  • 20+ Placeholders for use in scoreboards, holograms, and other plugins.
  • General: %factorycore_factories_owned%, _factories_running%, _total_value%, _highest_level%.
  • Invoice: %factorycore_invoices_pending%, _invoices_overdue%, _total_invoice_amount%.
  • Factory Type: %factorycore_steel_mills_owned%, _refineries_owned%, _workshops_owned%, _advanced_factories_owned%.
  • Per-Factory: %factorycore_factory_<id>_status%, _level%, _type%, _production%, _progress%.
+
Configuration Files
  • config.yml — Database, tax, salary, factory types, marketplace, notifications, GUI, debug, logging, production, economy, and placeholder settings.
  • resources.yml — Full definitions for all 21 resources with custom model data, material, name, lore, and glow.
  • recipes.yml — All 16 production recipes with inputs, outputs, timings, icons, and console commands.
  • language.yml — 80+ customizable messages, GUI titles, notification titles, bossbar text, and log formats.
  • npc.yml — NPC templates (4 types), employee shop listings (5 tiers), and global behavior settings (particles, sounds, holograms).
  • plugin.yml — Plugin metadata, full command definitions, and all permission node declarations.
+
Notification System
  • Title Notifications with configurable fade-in, stay, and fade-out durations for: Production Complete, New Invoice, Level Up, Factory Disabled, Tax Due, Overdue Invoice, Factory Purchased, Marketplace Sale.
  • Sound Effects: Production complete (ENTITY_PLAYER_LEVELUP), invoice created (BLOCK_NOTE_BLOCK_PLING), level up, tax due (BLOCK_NOTE_BLOCK_BASS), and marketplace sale (ENTITY_EXPERIENCE_ORB_PICKUP).
+
Logging System
  • 6 Log Categories: Admin commands, transactions, production, invoices, marketplace, and taxes — all individually togglable.
  • File-Based Logging to plugins/FactoryCore/logs/factorycore.log with 10MB max size and log rotation.
+
Developer API
  • FactoryCoreAPI Class providing full Java integration for third-party plugins.
  • Factory Management: getFactory(), getAllFactories(), createFactory(), removeFactory(), ownsFactory(), getFactoryStatus(), isProducing(), upgradeFactory().
  • Resource Management: getResource(), createResourceItem(), giveResource().
  • Storage Management: addToStorage(), removeFromStorage(), getStorageAmount(), getFactoryStorage().
  • Production & NPC: startProduction(), spawnNPC().
i
Technical Details
  • 54 Java Classes organized into: api/, commands/, gui/ (17 screens), listeners/ (5), managers/ (9), models/ (9), utils/ (6).
  • Build System: Maven with Shade plugin. Targets Minecraft 1.21.4, requires Java 17+.
  • Required Dependencies: Vault, WorldEdit 7.3.0, WorldGuard 7.0.9. Optional: PlaceholderAPI 2.11.5.
  • Database: YAML-based file storage in plugins/FactoryCore/data/.
  • API Version: Spigot/Paper 1.21.4. Author: aithor.
