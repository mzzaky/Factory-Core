Installation

Quick Start

Get FactoryCore up and running on your server in minutes.

üöÄ Installation Guide

Follow these steps to install and configure FactoryCore on your Minecraft server.

1

Install Dependencies

Make sure Vault, WorldEdit, and WorldGuard are installed and enabled on your server. An economy plugin (e.g. EssentialsX) is also required for Vault to function.

2

Drop the JAR

Place the FactoryCore.jar file into your server's /plugins/ folder.

3

Start / Restart the Server

Start or restart your server. FactoryCore will generate its configuration files inside plugins/FactoryCore/.

4

Configure & Define Factories

Edit config.yml to set tax rates, intervals, and other global settings. Define your factory types and recipes in the respective YAML files. Data is stored under plugins/FactoryCore/data/.

5

Read the Wiki

Dive into the Factory System, Employee System, and Tax System pages for detailed configuration and gameplay guides.
