Overview

FactoryCore provides extensive hook-ins for PlaceholderAPI (PAPI), allowing you to display real-time player data, factory statistics, and financial metrics on scoreboards, holograms, chat formats, or any plugins that support PAPI.

üí° Installation Requirement
To use these placeholders, you must have the PlaceholderAPI plugin installed on your server. Since our expansion is built internally, there is no need to download any eCloud expansion. It works straight out of the box!

Player Statistics

These placeholders display general information regarding the player's overall factory empire.

Placeholder Description Example Output
%factorycore_factories_owned% Total number of factories owned by the player. 5
%factorycore_factories_running% Number of player's factories currently running (producing items). 3
%factorycore_highest_level% The maximum level among all factories owned by the player. 4

Factory-Specific Data

You can retrieve data for a specific factory using its unique ID. Replace <id> with the factory's UUID.

Placeholder Description Example Output
%factorycore_factory_<id>_status% Current operational status of the factory (e.g., Running, Idle, Paused). Running
%factorycore_factory_<id>_level% Current level/tier of the specific factory. 2
%factorycore_factory_<id>_type% The structural type of the factory. Steel Mill
%factorycore_factory_<id>_production% The name of the recipe currently being produced by the factory. Iron Ingot
%factorycore_factory_<id>_progress% Current production progress percentage (0% to 100%). 45%

If the specified factory ID does not exist, or if the factory is not owned by the requesting player, the placeholders will return N/A or default zero values.

Finance & Taxes

Track the economic scale and tax burden of the player's empire.

Placeholder Description
%factorycore_total_value% The sum of base prices of all factories owned by the player, formatted to 2 decimals.
%factorycore_invoices_pending% The total number of unpaid tax invoices currently pending.
%factorycore_invoices_overdue% The total number of invoices that have missed their due date.
%factorycore_total_invoice_amount% The total combined amount owed across all pending invoices.

Specialized Factories

Count the specifics of what types of factories make up the player's industrial infrastructure.

Placeholder Description
%factorycore_steel_mills_owned% Number of Steel Mills owned.
%factorycore_refineries_owned% Number of Refineries owned.
%factorycore_workshops_owned% Number of Workshops owned.
%factorycore_advanced_factories_owned% Number of Advanced Factories owned.

Developer Tips

Are you a developer hooking into FactoryCore? Here are some crucial tips for managing PlaceholderAPI optimally in your custom solutions.

‚ö°

Async Resolution

Whenever you are parsing PlaceholderAPI strings (using PlaceholderAPI.setPlaceholders()), ensure you avoid doing it purely on the main Server thread if you are fetching thousands at once. Although FactoryCore caches logic efficiently, heavy operations should run async.

üîå

Direct API vs PAPI

If you are writing an add-on or plugin dedicated to FactoryCore, prefer using the FactoryCore.getInstance().getFactoryManager() methods directly rather than parsing strings via PAPI. Direct method calls are significantly faster and type-safe.

üß†

Stream Mapping

Notice that placeholders like %factorycore_total_value% loop and map total stream values. If you are polling this frequently in scoreboards every tick, cache the string result temporarily instead of re-calculating on loop.

Always check if the PlaceholderAPI plugin is installed before executing parsing logic in your dependent plugin to avoid NoClassDefFoundError.
